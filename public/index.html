ChainVault Backend API

Welcome to the ChainVault backend API. This service provides authentication endpoints for X (Twitter) integration.

Available Endpoints

X Authentication Start

GET /api/auth/x-start

Initiates the X (Twitter) OAuth 2.0 authentication flow.

X Authentication Callback

GET /api/auth/x-callback

Handles the OAuth callback from X and exchanges the authorization code for an access token.
